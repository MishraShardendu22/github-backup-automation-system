# logs

## Endpoint

- URL: https://github-backup-script.onrender.com/api/logs?page=1&limit=100&level=INFO
- Method: GET
- Query Params: page=1, limit=100, level=INFO
- Description: Execution logs

## Pagination

```json
{
  "page": 1,
  "limit": 100,
  "total_items": 0,
  "total_pages": 1
}
```

## Full Response

```json
{
  "data": [],
  "pagination": {
    "page": 1,
    "limit": 100,
    "total_items": 0,
    "total_pages": 1
  }
}
```
